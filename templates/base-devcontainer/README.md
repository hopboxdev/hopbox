# hopbox devcontainer base image

Published as `ghcr.io/hopboxdev/devcontainer-base:<version>`. Used as the default `image` in new `devcontainer.json` files generated by hopbox.

## Build locally

From repo root:

    make base-image

Builds `ghcr.io/hopboxdev/devcontainer-base:dev` into the local Docker daemon.

## Publish

    docker tag ghcr.io/hopboxdev/devcontainer-base:dev ghcr.io/hopboxdev/devcontainer-base:<version>
    docker push ghcr.io/hopboxdev/devcontainer-base:<version>
